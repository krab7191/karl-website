# Karl Rabe portfolio website

- [MetaMindWorks.com](https://metamindworks.com)

## Tech

- Node + NPM
- Vite
- Svelte
- Routify
- Flowbite CSS

- Hosted on [Netlify](https://www.netlify.com/)

### Development instructions

1. Install Node + NPM. Versions in package.json "engines" key
2. Pull source code
3. Run `npm i`
4. Run `npm run dev`
